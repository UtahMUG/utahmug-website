Version 9.2.0 to the Wasatch Front Travel Demand Model (WF TDM) has been released. To access the new model, please visit the [WF TDM Official Releases Repository](https://github.com/WFRCAnalytics/WF-TDM-Official-Releases/releases/tag/v9.2.0-official){:target="_blank"}. If do not have access to the repository please request access by emailing Suzie Swim at sswim@wfrc.utah.gov or Tim Hereth at thereth@mountainland.org.

The online documentation, including the What's New Document and Model Validation Report, can be accessed [here](https://wfrc.utah.gov/wftdm-docs/v9x/v920/).

As stated in the document, version 9.2.0 of the Wasatch Front Travel Demand Model includes key updates in the network inputs. The highway network was revised to reflect WFRC’s Amendment #4 to the 2023–2050 RTP, incorporating changes to Legacy Parkway and minor changes to I-15 in Box Elder County.

Have a great day!
